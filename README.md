# Ping

A simple ping docker container.

# Configuration

| Env var name  | Description                                                                         |
|---------------|-------------------------------------------------------------------------------------|
| PING_TARGET   | Define the target address to be pinged, if not defined, the default gateway is used |   
| PING_INTERVAL | Set the ping interval (default 1s)                                                  |
